Welcome to L!Tabs.

Light Weigh Tabs for geeks, Deprecated script from 2014, bring to live. Before jQuery Tabs, We had L!Tabs, a small script that will allow you to create beautiful already coded tabs. Download the src and included to your project and let the code take the heacy lifting. Easy modification for your own benefits.

Templates Included

Weâ€™ve two templates included with this script, classic view, and a modern blue skin view, depending on your taste. To switch through skins please add the proper css called.

For a classic skin use:

<link href="css/litabs.min.css" rel="stylesheet" />

For a modern skin feel use:

<link href="css/litabs-modern.min.css" rel="stylesheet" />

To avoid platform problems always include:

<link rel="stylesheet" media="only screen and (max-width: 400px)" href="css/litabs-mobile.css" />

Installation & Usage

  1. Download source code and copy from src to your project folder.
  2. Make the CSS & JS calls for script dependencies.
<link href="src/litabs.min.css" rel="stylesheet" />
<link rel="stylesheet" media="only screen and (max-width: 400px)" href="css/litabs-mobile.css" />
<script src="src/litabs.min.js"></script>

<script>
    $(function() { $(".tabs").Litabs(); });
</script>
